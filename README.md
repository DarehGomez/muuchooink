# Proyecto Universitario  
## Empresa: MUUCHO OINK

### Descripción
Este proyecto académico se desarrolló como parte de las materias de **Programación Orientada a Objetos (POO)**, **Bases de Datos** y **Tópicos de Software**. El objetivo fue implementar una solución tecnológica adaptada a la empresa **MUUCHO OINK**, integrando funcionalidades prácticas y un enfoque profesional.

### Objetivos
- **Aplicar POO** en el diseño y desarrollo de la aplicación.  
- **Diseñar bases de datos** seguras y funcionales para la gestión de información.  
- **Implementar tópicos de software** en la construcción de un sistema adaptable y escalable.  

### Modificaciones Realizadas
El proyecto se basó en un repositorio público de referencia, al cual se le realizaron cambios para adaptarlo a las necesidades de **MUUCHO OINK**, entre ellos:  
- Ajustes en **assets** y recursos gráficos.  
- Implementación de un **sistema de inicio de sesión** para usuarios.  
- Adaptación de la estructura a los requerimientos académicos y empresariales.  

### Créditos
Proyecto realizado por estudiantes universitarios, con base en conocimientos adquiridos en el área de desarrollo de software. Adaptado y mejorado para la empresa **MUUCHO OINK**, garantizando un enfoque original y profesional.
